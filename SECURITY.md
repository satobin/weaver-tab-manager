# Security Policy

## Supported Version

Security fixes target the latest version published through the Chrome Web Store.

## Reporting A Vulnerability

Report a suspected vulnerability privately to [weavertabmanager@gmail.com](mailto:weavertabmanager@gmail.com). Do not disclose vulnerability details or private browsing data in a public issue.

Include the affected Weaver version, browser name and version, impact, and minimal reproduction steps. Reports will be acknowledged after review; timing for a fix or disclosure depends on severity and scope.
