# Security Policy

## Supported Version

Security fixes target the latest version published through the Chrome Web Store.

## Reporting A Vulnerability

Use [GitHub private vulnerability reporting](https://github.com/satobin/weaver-tab-manager/security/advisories/new)
to report a suspected vulnerability. Do not disclose vulnerability details or private browsing data
in a public issue.

Include the affected Weaver version, Chrome version, impact, and minimal reproduction steps. Reports
will be acknowledged after review; timing for a fix or disclosure depends on severity and scope.
