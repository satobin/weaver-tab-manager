# Weaver Support

Use [GitHub Issues](https://github.com/satobin/weaver-tab-manager/issues) to report a reproducible
bug or request a feature. The Chrome Web Store Support hub will also be available after the first
release is published.

Include the Weaver version, Chrome version, operating system, expected behavior, and the shortest
set of steps that reproduces the problem.

Do not include private tab titles, URLs, saved-window names or contents, screenshots containing
browsing data, authentication information, or other personal information. Replace those values with
neutral examples before posting.

For a suspected security vulnerability, follow [SECURITY.md](SECURITY.md) instead of opening a
public issue.
