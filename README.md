# Weaver

**Less tab chaos. More focus.**

Weaver gives you a clear view of every browser window, helps you find the tab you need, and lets you save a window to restore later.

![Weaver showing every browser window in one clear view](assets/readme/weaver-every-window.jpg)

## Install

[Install Weaver from the Chrome Web Store](https://chromewebstore.google.com/detail/weaver-window-tab-manager/lchcjicakojjacjpleolmjcjlppaeobn). Weaver works in Chromium-based browsers that support Chrome Web Store extensions.

[View Weaver's source code on GitHub](https://github.com/satobin/weaver-tab-manager).

## Features

- Search, sort, move, merge, suspend, restore, and close tabs across windows.
- Collapse window cards to hide tab rows when you want a cleaner overview.
- Save named window snapshots locally and restore them later.
- Close exact duplicate URLs, with optional matching for Google Docs, Sheets, Slides, Notion, and user-defined sites.
- Preserve browser tab groups while organizing tabs.
- Use System, Light, or Dark appearance modes.

Advanced duplicate matching is off by default. Exact full-URL duplicate matching remains available without enabling site-specific rules.

## Privacy

Weaver has no account, analytics, or advertising. Open-tab details, preferences, custom rules, and saved-window snapshots are processed locally in your browser and are not sent off your device. See the [privacy policy](PRIVACY.md) for the full disclosure.
